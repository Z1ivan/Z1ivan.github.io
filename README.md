# Z1ivan Homepage

Personal academic homepage for **Lifan Zheng**.

## Local Preview

```bash
bash run_local_preview.sh
```

Then open: `http://127.0.0.1:4000/`

## Main Content Files

- `_pages/about.md` (English homepage)
- `_pages/about_zh.md` (Chinese homepage)
- `_data/publications.yml` (publications)
- `_data/projects.yml` (projects)
- `assets/css/main.scss` (custom styles)

## Notes

- Site is based on Jekyll.
- Main branch is deployed to GitHub Pages.
